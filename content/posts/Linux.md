
### Linux 系统

#### 基本的Linux命令

`ls` 显示当前目录文件 list directory
`pwd` 显示当前路径 present working directory
`cat` 将文件列出 concatenate    
`less` or `more` 查看当前文件
`cd` 改变路径 change directory
`mkdir` 创建目录 make directory
`mv` 将文件和目录从一个地方转移到另一个地方 重命名
`top` 查看性能
`man` 工具书 manual
`tail` 查看后几个文件
`rm` 删除文件 remove
`vi` 编辑文件

`vi` + filename 直接打开或创建文件
进入vi编辑器之后，按`i`可以进入编辑（insert）模式，`Esc`可以退出编辑模式，进入编辑模式
输入`:wq`就是保存并且退出
输入`:q!`就是不保存然后退出
